# LocalMapReduce

A simple local MapReduce simulation using Python and LevelDB (via Plyvel).

## Project Structure
- `data/` — Input text files
- `db/` — LevelDB storage
- `output/` — (optional future use)
- `scripts/` — Helper scripts

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
